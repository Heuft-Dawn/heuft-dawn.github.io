Topic 9

Standard JavaScript Events Including those for Mobile Devices ( Ex. onTouchBegin, onLoad, etc.) and Animation and Transition Events


Here is a sample list of common HTML events that JavaScript can be combined with to execute code.

  • onclick
  • onchange
  • onload
  • onmouseover
  • onmouseout
  • onkeydown

In addition, here are a few events that are related to mobile devices. These execute when a finger interacts with a DOM event.

  • touchstart
  • touchmove
  • touchend
